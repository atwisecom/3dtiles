# 3dtiles
tool for convert 3d models to 3dtiles format

(supports Cesium and Skyline TerraExplorer)

input: 
 - gltf, glb 
 - 3ds max 
 - fbx 
 - osgb
 - dae
 - obj
 - rvt (autodesk revit) 
 - ifc 
 - dgn (Bentley microstation) 
 - eff (ctech.com EVS Field file formats) 
 - txt, las (point cloud)
 - x
 - ...
